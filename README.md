

# 📘 Student Form Web Application

## 📌 Overview

This project is a simple **Student Form Web Application** built using **HTML, CSS, and JavaScript**. It allows users to input student details, display them dynamically, and remove entries when needed.

The main goal of this project is to demonstrate basic **DOM manipulation**, **event handling**, and **dynamic data management** using JavaScript.

---

## 🚀 Features

* ➕ Add student details through a form
* 📋 Display entered student data instantly on the page
* ❌ Remove individual student entries
* ⚡ Real-time updates without page reload
* 🎨 Styled using CSS for a clean and user-friendly interface

---

## 🛠️ Technologies Used

* **HTML** – Structure of the form and layout
* **CSS** – Styling and design
* **JavaScript** – Functionality (Add & Remove operations)

## 🧠 How It Works

### 1. Adding Student Details

* User fills out the form fields (e.g., name, age, course, etc.)
* When the **Add button** is clicked:

  * JavaScript captures the input values
  * A new row/item is created dynamically
  * The data is displayed on the page

### 2. Removing Student Details

* Each added student entry includes a **Delete button**
* When clicked:

  * JavaScript removes the specific entry from the DOM

---

## 📸 Example Workflow

1. Enter student details in the form
2. Click **Save**
3. See the student appear in the list/table
4. Click **Delete** to remove the student
